MomentumShotz

MomentumShotz is a small, single-operator automated tool that curates short motivational segments from long-form Creative Commons-licensed speeches and talks on YouTube, edits them into vertical short-form clips (captions, framing, on-screen creator credit), and publishes them to YouTube Shorts and TikTok.

Every clip carries an on-screen credit to the original creator, and every post description includes an attribution line and a link back to the source video, in line with the Creative Commons license each source video is published under. Only videos explicitly marked creativeCommon by their uploader are ever used.

This is not a raw repost tool: each clip is a curated, re-edited excerpt (60–90 seconds, selected and captioned) of a longer original work, not a copy of the full video.
